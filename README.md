# JPC Portfolio Website

A modern, responsive portfolio website built with React, TypeScript, and Tailwind CSS.

## 🛠️ Technologies

- React
- TypeScript
- Tailwind CSS
- Vite
- Bun

## 🏗️ Installation

1. Install dependencies:

```console
$ bun install
```

2. Start the development server:

```console
$ bun dev
```

#. Build for production:

```console
$ bun build
```
